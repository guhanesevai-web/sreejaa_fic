
## 🎯 Features in Detail

### Owner Dashboard
- Complete company profile management
- Add/Edit customer details
- View all loans and collections
- Export reports to Excel
- Date-wise collection filtering

### Staff Dashboard
- Customer management (view only)
- Process interest payments
- Generate collection reports
- Print bills
- View collection summaries

### Payment Rules
- ✅ **Only Interest is collected**
- ❌ **Principal amount NEVER reduces**
- Weekly/Monthly due collection
- Automatic receipt generation

## 📊 Report Export

Click **"Export to Excel"** button to download:
- Weekly Collection Report
- Monthly Collection Report
- Date-filtered Reports

## 🖨️ Bill Printing

Each collection includes a print option:
- Professional bill format
- Company logo support
- Automatic print dialog
